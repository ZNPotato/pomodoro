# pomodoro
기능 : 40분 집중 - 20분 휴식 기능 (시간 조정 불가)
(탭 설명) 
할일 : 해야할 것 작성 및 시작-종료를 통해 타임라인 남길 수 있음
카테고리 : 할일의 내용들의 카테고리 설정 가능
기록 : 당일 카테고리별 할애한 시간, 집중 총 시간 등 확인 가능 (과거 타임라인 확인 가능)
자동 저장 : Windows 운영체제일 경우 json 파일을 저장하여 노트북 <-> 데스크탑 에 기록 유지하며 사용가능 (단, 저장 -> .json 파일 옮김 -> 불러오기 해야함)
